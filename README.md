A lightweight macOS-style practice ticketing system (Electron + Vite + React).

## Features

- Create and list tickets
- macOS system notifications when creating tickets
- Minimal scaffold for adding persistence (lowdb)

## Quickstart

Requirements: Node 18+, npm, macOS (for notifications and best Electron support)

Install and run in development mode:

```bash
cd mac-ticketing
npm install
npm run dev
```

The app will start Vite at `http://localhost:5173` and launch an Electron window.

## Packaging

This scaffold doesn't include packaging configuration yet. For distribution, add `electron-builder` or similar.
